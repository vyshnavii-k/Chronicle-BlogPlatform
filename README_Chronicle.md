# 📖 Chronicle

A full-stack blogging platform where users can log in, write and publish blog posts, and read stories shared by others — all in one clean, minimal space.

🔗 **GitHub:** [github.com/vyshnavii-k/BlogPlatform](https://github.com/vyshnavii-k/BlogPlatform)

---

## ✨ Features

### 🔐 User Authentication
- Secure login and logout
- Only authenticated users can publish blog posts
- Public reading — anyone can browse and read posts

### 📝 Blog Management
- **Write & Publish** blog posts with a clean editor
- **Read** posts shared by the community
- Organized feed of published blogs

## 🚀 Getting Started

### Prerequisites
- Node.js v16+

### Installation

```bash
# Clone the repository
git clone https://github.com/vyshnavii-k/BlogPlatform.git
cd BlogPlatform

# Install dependencies
npm install

# Start the server
node server.js
```

Then open `http://localhost:3000` in your browser.
